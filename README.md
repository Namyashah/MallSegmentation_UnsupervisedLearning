# 🧠 Mall Customer Segmentation using Unsupervised Machine Learning

A comprehensive implementation of Unsupervised Machine Learning
techniques to analyze customer behavior, discover hidden patterns,
detect anomalies, and build recommendation systems using the Mall
Customer Dataset.

------------------------------------------------------------------------

## 📌 Project Overview

This project explores multiple unsupervised learning algorithms to
perform:

-   🎯 Customer Segmentation
-   📊 Cluster Evaluation
-   🌳 Hierarchical Clustering
-   📍 Density-Based Clustering (DBSCAN)
-   📉 Dimensionality Reduction
-   🚨 Anomaly Detection
-   💡 Recommendation Systems

------------------------------------------------------------------------

## 📂 Dataset Features

-   Customer ID
-   Gender
-   Age
-   Annual Income
-   Spending Score
-   Purchase Frequency
-   Recency

------------------------------------------------------------------------

## 🚀 Techniques Implemented

### 🔹 Clustering

-   K-Means Clustering
-   Elbow Method
-   Silhouette Score
-   Davies-Bouldin Index
-   Calinski-Harabasz Index

### 🔹 Hierarchical & Density-Based Clustering

-   Agglomerative Clustering
-   Dendrogram Analysis
-   DBSCAN
-   K-Distance Graph

### 🔹 Dimensionality Reduction

-   Principal Component Analysis (PCA)
-   Kernel PCA (RBF Kernel)
-   t-SNE Visualization

### 🔹 Anomaly Detection

-   Isolation Forest
-   Local Outlier Factor (LOF)
-   One-Class SVM (PyOD)

### 🔹 Recommendation Systems

-   Content-Based Recommendation (Cosine Similarity)
-   Cluster-Based Recommendation
